# Exercícios-estágio
Exercícios feitos em Java, minha linguagem principal de estudo. Os códigos estão organizados dentro da pasta src.

Em todas as funções, foram definidos limites de execução para evitar erros de StackOverflow, garantindo a estabilidade das chamadas recursivas.

No cálculo da sequência de Fibonacci, utilizei a classe BigInteger para contornar a limitação de tamanho dos tipos primitivos inteiros do Java. Mesmo com essa flexibilidade, estabeleci um limite máximo de processamento para prevenir o consumo excessivo de memória e lentidão do sistema..
